# 펀드투자권유 자문인력

## 제1과목 펀드일반
1. 투자펀드의 특징
  - 집단성,간접성, 실적배당원칙, 투자평등의 원칙, 펀드 자산의 분리(신탁과 회사제도 이용)

2. 수익자총회
  - 의결사항 : 합병, 환매연기, 신탁계약 변경
    - 집합투자업자, 신탁업자, 신탁계약기간, 투자신탁 종류, 주된 투자대상, 투자자산의 투자한도 변경
    - 보수, 수수료 인상, 개방형을 폐쇄형으로 변경, 환매대금지급일 연장
  - 집합투자업자 소집 원칙 : 신탁업자, 5% 이상 요구 가능
  - 의결 : 출석수익자 과반수, 전체 좌수의 1/4
    - 신탁계약에 정한 사항 과반수와 1/5, 서면 의결행사 가능
  - 연기수익자총회 : 2주 이내, 과반수와 1/8, 1/10
  - 간주의결권 : 통지 있었으나 의결권 미행사, 행사방법 규약에 기재, 총좌수의 1/10이상 의결, 간주의결권행사 결과 공시
  - 반대매수 : 청구기간 20일

3. 증권신고서, 투자설명서
  - 증권신고서 : 무액면, 기명식, 예탁발행
    - 매출 및 모집금액 10억원 이상, 효력발생 15일, 상장폐쇄형 10일, 비상장폐쇄형 7일, 정정신고 3일
    - 개방형펀드는 일괄신고 가능
  - 등록신청서 : 출자 및 자본금 1억원 이상, 20일내 등록 여부결정, 변경등록 2주 이내
    - 증권신고(정정신고) 효력발생 -> 등록(변경등록)으로 간주
  - 투자설명서 : 증권신고 효력발생일에 제출
    - 법정권유문서 : 투자설명서에 의해서만 권유 가능
    - 정식(효력 후), 예비(효력 전), 간이(효력 전/후)
    - 매년 1회 이상 갱신, 변경등록 후 5일 이내 갱신

4. 판매수수료와 판매보수
  - 판매수수료 : 일회성, 투자자가 납부, 2.0%
  - 판매보수 : 지속성, 펀드가 납부, 1.0%, 이연판매보수 1.5%

5. 부당권유금지
  - 거짓
  - 불확실
  - 불초청권유(장외파생상품)
  - 거부의사(1개월 경과 후, 다른 종류)
  - 금전

6. 펀드의 환매
  - 대금지급 : 청구일로부터 15일
    - 시장성 없는 자산에 10%, 해외자산에 50% 초과
    - 사모투자, 부동산, 특별자산 재간접 펀드
  - 환매연기 : 6주 이내 총회의결, 부분환매연기, 펀드분리 가능

7. 펀드의 운용제한
  - 동일 종목 증권 : 10%
    - 100% : 국채, 통안채, 정부보증채
    - 30% : 지방채, 특수채, OECD, 파생결합증권
    - 25% : 50% 이상을 5%씩 나누어 투자
    - 시가총액비중 10% 초과 시 그 시가총액 비중
  - 동일 법인이 발행한 증권 총수 : 10%, 운용사 전체 : 20%
  - 파생상품 운용제한
    - 적격 거래상대방 외 장외파생상품 거래 금지(사모도 적용)
    - 위험평가액 순자산총액 100%(사모 400%)
    - 동일 기초자산 10%, 동일 거래상대방 10%
  - 부동산 : 취득 후 처분금지(예외 인정, 사모도 적용)
    - 국내 소재 1년, 국외 소재 규약이 정한 기간
  - 다른 집합투자증권 : 동일 운용사 50%
    - 동일 펀드 20%, 동일 펀드증권 20%
    - 재간접형펀드, 사모펀드에 대한 투자금지
  - RP매도 50%, 증권대여 50%, 증권차입 20%

8. 투자광고
  - 반드시 포함 사항 : 권고, 수익, 위험

9. 자산운용보고서
  - 집합투자업자가 작성 : 작성, 교부비용 집합투자업자 부담
  - 3개월에 1회 이상 제공 : 직접, 전자우편 교부 가능

10. 단기금융집합투자기구
  - 단기상품
    - 잔존만기 6개월 이내인 양도성예금증서, 잔존만기 5년 이내인 국채, 잔존만기 1년 이내인 채권 및 기업어음, 만기 1년 이내인 어음
    - 단기대출, 6개월 이내 금융기관 예치, 다른 MMF증권
  - 운용제한
    - 추가설정 : 개인용 3천억, 법인용 5천억
    - 증권의 대여 또는 차입 금지
    - 만기 1년 이상 국채, RP매도 : 5% 이내
    - 가중평균 잔존만기 : 75일 이내
    - 채권운용 : 40% 이상, 신용등급 상위 2개
      - 동일채권 최상위 5%, 차상위 2% 이내
      - 동일어음 최상위 3%, 차상위 1% 이내
    - 장부가평가 : 괴리율 0.5% 이상 시가평가
    - 당일출금(자기거래) : 급여이체, 금융투자상품의 환매, 자동이체, 금융투자상품으로의 이체
      - Max[판매규모 5%, 100억원]

11. 특수한 형태의 집합투자기구
  - 환매금지형
    - 환매금지 : 설정일로부터 90일 이내 상장의무
    - 의무 : 부동산/특별자산/혼합자산펀드, 시장성 없는 자산 20%초과
    - 추가발행 : 신탁업자 확인, 투자자 전원 동의, 이익분배금 범위 내, 기존 투자비율에 따른 추가 매수기회 부여 시 가능
    - 존속기간 정한 펀드로 설정
    - 기준가격 산정 및 공고 의무 적용배체
  - 종류형
    - 판매수수료, 판매보수 외 차등 금지
    - 특정 종류의 집합투자자만으로 총회개최 가능
    - 비종류형을 전환 가능 (포트폴리오 변동 없음)
  - 전환형
    - 수수료 없이 펀드전환 가능 (포트폴리오 변동)
    - 펀드의 법적형태 동일
  - 모자형
    - 운용의 효율성이 목적 : 규모의 경제 효과
    - 자펀드는 모펀드 이외의 펀드에 투자 금지
    - 모펀드에는 자펀드 이외의 펀드 투자 금지
    - 집합투자업자 동일, 투자자에게 자펀드를 판매
    - 기존펀드의 모자형펀드로의 변경 가능, 단순 합병, 병합 금지
    * 재간접펀드 : 운용능력 아웃소싱, 상위펀드 투자, 집합투자업자 상이
  - 상장지수펀드
    - 인덱스펀드, 주식처럼 거래
      - 설정일로부터 30일 이내에 상장의무
    - 설정일로부터 30일 이내에 상장의무
    - ETF 설정 의한 이해관계인과의 거래 허용
    - 동일종목에 30%, 동일법인에 20% 운용 가능

12. 임의해지와 법정해지
  - 법정해지
    - 신탁계약기간 종료, 피흡수합병, 등록취소
    - 수익자총회의 해지 결의, 수익자총수 1인이 되는 경우
    - 투자신탁인 전문투자형 사모펀드의 해지명령
  - 임의해지
    - 전원의 동의, 수익증권 전부의 환매청구
    - 1년 되는 날 투자신탁 원본액 50억원 미달
    - 1년 이후 1개월간 계속해서 50억원에 미달
  * 외부회계감사 : 자산총액 300억 이하 또는 300~500억이고 6개월간 추가발행 없는 경우 면제

13. 금융소비자보호에 관한 법률
  - 금융소비자보호의 필요성 : 비경쟁적 산업구조, 협상력의 열위, 정보의 비대칭성, 상품구조, 복잡한 금융상품
  - 기능별 규율 : 금융상품(예금성, 투자성, 보장성, 대출성), 판매업자(직접판매업자, 판매대리, 중개업자, 자문업자)
  - 6대 판매원칙 : 적합성원칙, 적정성원칙, 설명의무, 불공정 영업행위 금지, 부당권유행위 금지, 허위,과장광고 금지
  - 위반시 제재 강화 : 위법계약 해지권(5년), 판매제한명령, 손배배상 입증책임 전환(금융회사), 징벌적 과징금(50%), 과태료(적합성/적정성원칙 3천만원, 설명의무/불건전영업행위/부당권유/광고규제 1억원)
  - 권익보호 제도 : 청약철회권(보장성 증권수령일부터 15일, 투자성 계약체결일부터 7일, 대출성 계약일부터 14일), 사후 구제 강화(분쟁조정 이탈금지, 법원의 소송중지, 소비자의 금융회사에 대한 자료요구)
  - 기타 : 금융상품 자문업 신설(판매와 자문 겸영 금지), 금융교육 강화, 직판업자의 관리책임, 대출모집인 감독 강화

14. 이해상충의 방지의무
  - 충실의무 : 최선의 집행의무, 수단채무
  - 원인 : 정보의 비대칭, 겸영업무 범위 확대
  - 이해상충 회피 : 공시 - 저감(낮추기) - 거래단념
  - 정보교류차단, 조사분석자료 작성,제공 제한
  - 자기거래 금지 : 공개시장에서의 거래 제외
  - 이익의 우선순위 : (고객=고객) > 회사 > 임직원

15. 상품 판매 단계의 금융소비자보호
  - 적합성의 원칙 : 투자자구분 -> KYCR -> 투자자확인 -> 부적합한 투자권유 금지
    - 전문투자자 요구시 일반투자자 대우 허용(반대는 금지)
    - 상대적 전문투자자 : 장외파생상품 거래시 주권상장법인 일반투자자로 간주, 요청시 전문투자자로 대우
    - 자발적 전문투자자 : 100억원 이상인 법인, 직전 5년 중 1년 이상 5천만원 이상인 개인(소득 1억원, 부부합산 1억 5천만원 / 자산 5억원 / 전문성 전문직종 1년 이상)
  - 적정성의 원칙 : 투자권유하지 않고 파생상품 등 판매시
  - 설명의무 : 상품 및 고객 따라 차등 설명 가능, 이해하였음을 확인, 손익구조 및 손실위험 이해불가시 투자권유 중단
    - 설명의무 위반 시 배상책인(원본손실액 배상)
  - 손실보전 금지 : 권한 남용 금지
  - 합리적 근거의 제공 : 출처, 제공자 명시
  - 적정한 표시의무 : 중요사실 표시(방법은 불문), 투자성과 보장 금지, 허위,과자,부실 표시 금지
  - 불초청권유 금지 : 장외파생상품
    - 거부 표시 후 재권유 : 1개월 경과 후, 다른 종류 가능

16. 상품판매 이후 단계의 금융소비자보호
  - 보고 및 기록의무 : 처리결과의 보고(매매명세서 통지), 기록 및 증거유지, 일임매매(5년 이하 직역, 2억원 이하 벌금)
  - 정보의 누설 및 부당이용 금지 : 강행법 위반
  공정성 유지 의무 : 투자정보 차별제공 금지
  * 해피콜(7일 이내, 다른 부서), 미스터리 쇼핑, 불완전판매 배상제고 (15일 이내), 판매수수료 반환서비스(5일 이내))

17. 본인에 대한 윤리
  - 법규준수 : 몰라도 처벌, 법정신과 취지 포함
  - 자기혁신 : 전문지식 배양의무
  - 품위유지 : 독립성유지 의무(부당지시 거절)
    - 문화활동 목적의 상품권은 수수 가능
    - 거래상대방만 참여한 여가활동 등 비용 제공 금지
    - 수수금액 10억원 초과 시 홈페이지 공시
  - 직무관련 정보의 사적거래 금지 : 미공개중요 정보의 이용
  - 직위의 사적이용 금지 : 경조사 등은 예외

18. 회사에 대한 윤리
  - 공용재산의 사적사용 및 수익금지
  - 경영진의 책임 : 사용자의 책임, 관리감독책임
  - 정보보호 : 불명확시 비밀정보로 분류,취급
  - 내부제보제도 : 제보자 비밀보장, 신분상 불이익 금지
  - 고용계약 종료 후 의무 : 일정기간 지속
    - 지적재산물의 사용 및 이용권 회사 귀속

19. 대외활동 시 준법절차
  - 사전 승인 후 활동 : 회사가 요구 시 즉시 중단
  - 온라인커뮤니티, SNS 등도 대외활동에 포함
  - 고객과의 미레일은 장소불문 절차 준수
  - 금융투자상품 관련 내용은 사전 승인 후 게시
    - 예외 : 기술적분석, 출처를 명시한 인용

20. 사회 등에 대한 윤리
  - 시장질서 존중 : 대상자 및 대상정보 확대
    - 시장실서 교란횡의 : 5억원 이하 과징금
    - 이익,손실회피 금액의 1.5배가 5억원 초과시 그 금액

36. 주가배수를 이용한 상대가치 평가방법
  - PER
    - 주가가 주당순이익의 몇 배인가를 평가
    - PER=p/EPS, P-EPS*PER
  - PBR
    - 주가가 주당순자산의 몇 배인가를 평가
    - PER은 유향, PBR은 저량 관계
    - 주가 = PBR * 주당순자산
  - PSR
    - 주가가 주당매출액의 몇 배인가를 평가, 왜곡,조작가능성 가장 낮음
    - 주가 = 적정P/S*주당매출액
  - EV/EBITA
    - 기업가치(EV)와 이자,세금,감가상각 전 이익(EBITA)의 비율
    - 현금흐름에 대한 기업의 상대가치
    - 경기상황 악화 주가하락 시 운용

  * 적극적 주식운용전략 : 가치주투자, 성장주투자, 시장투자(인덱스+a), 중소형주투자전략
  * 소극적 주식운용전략 : 인덱싱전략

38. 펀드의 위험과 수익율
  - 펀드 분석은 특성을 찾는 과정, 펀드 평가는 우열이나 순위를 가려 재추자 여부 결정
  - 절대위험 : 표준편차, 최대손실예상금액(VaR)
  - 상대위험 : 공분산, 초과수익률, 상대VaR, 베타, 추적오차
  - 수익률 : 중도 분배 있을 경우 분배율 고려햐여 산출
  - 운용사 그룹수익률 : 대표계정 및 생종계정오류 제거, 운용사간 성과비교 가능
  - 벤치마크는 절대평가, 유형평균은 상대평가
    - 2이상의 지수를 합성한 합성지수도 벤치마크로 사용 가능

39. 위험조정성과
  - 샤프 : (Rp-Rf)/g, 위험 단위당 수익, 높을수록 우수
  - 알파 : (Rp-Rf)-(b*(Rm-Rf)), 운용역량평가, 종목선택과 시장예측 성과 구분 곤란
  - 트레킹에러 : 벤치마크 수익률 대비 오차
  - 정보비율 : (Rp-Rf)/TE, 추적오차 단위당 수익, 추적오차가 수익에 기여한 정도

## 재2과목 파생상품펀드
1. 위험평가액
  - 선물(선도) : 선물가격 * 계약수 * 승수
  - 옵션 매수 : 선물가격 * 계약수 * 승수 * 델타 : 델타위험액
  - 옵션 매도 : 델타위험액 + 감마위험액 + 베가위험액
  - 통화스왑 : 지급하기로 한 통화의 명목원금
  - 금리스왑 : 고정금리지급 시 만기까지 지급액
    - 변동금리지급 시 예상 지금액의 시가평가액
  - 신용부도스왑 : 보장매수자는 지급총액
    - 보장매도자는 사건발생시 지급해야 할 명목금액
  - 총수익스왑 : (+)수취 시 지급금액
    - (-)수취 시 지급금액 + 수취금액의 절대값,
  - 기초자산의 의 교환을 포함하는 스왑거래
    - 기초자산가격 + 지급하기로 한 금전총액
  - 기초자산을 제외한 금전만 교환하는 스왑거래
    - 지급하기로 한 금전총액
  - 선도,옵션,스왑 거래의 혼합 : 각 위험평가액 산정방법 준용
  - 최대손실예상금액의 제한 있는 경우 최대손실예상금액
  - 장외파생상품 : 합의한 명목원금

2. 공시대상위험지표
  - 계약금액 : 매수, 매도, 순포지션 구분
  - 만기 손익구조 : 이익구간, 0 구간, 손실구간
  - 손익구조변동 : 시나리오법 (Stress test)
  - 최대손실예상금액 : 10영업일, 99% 신뢰구간
    - VaR = 포지션 시장가치 * 신뢰구간별 표준편차의 배수 * 포지션의 표준편차 * 보유기간(루트)
  - 계약금액, 만기 손익구조는 거래 다음날, 손익구조변동, 최대 손실예상금액은 매일 공시

3. 워런트 투자형
  - 특징 : 레버리지 효과, 원금보존 추구형 가능, 가격위험과 신용위험에 동시에 노출
  - 콜(상승형), 풋(하락형),
  - 디지털(받거나 받지 못하는 구조), 스프레드(특정 구간을 넘어서면 일정 수익), 레인지(구간에 잇을 때만 수익), 낙아웃, 낙인(표준옵션 보다 저렴)
  - 유럽형(만기에 한번), 미국형(만기 전 아무 때나), 아시아형(평균가격 사용, 상품연계), 참여율, 리베이트(낙아웃 시 지급)

4. 장내파생상품 운용형
  - 인덱스 : 리버스/인버스, 레버리지, 인핸스드
  - 옵션복제 : 풋옵션 매도포지션의 수익구조 복제, 리버스컨버터블(풋매도), 포트폴리오보험(콜매수)
  - 델타복제 : 풋옵션 매도의 델타를 참조

5. 주요 파생상품펀드
  - 금리연계 파생상품펀드 : 위험 헤지 목적이 대부분
    - 만기길고, 변동성 작아 공모펀드 곤란
    - 발행자에 중도상환 권리 부여
  - 상품연계 파생상품펀드 : 인플레이션 헤지 효과
    - 다른 자산과 상관관계 낮아 분산투자 효과 기대
    - 가격 변동선/변동폭 큼, 아시아형옵션 사용
    - 선물로 거래 : 만기이월 영향, 추가수익 가능
    - 상품인덱스 : 변동성 커 액티브적 요소 가미
  - 멀티에셋 : 안정적,보수적 운용전략

6. 포트폴리오보험형 펀드
  - 콜옵션 매수성과 복제
    - 방어적 풋 : 주식 매수, 풋옵션 매수, 비용 비쌈
    - 이자추출 : 채권 매수, 콜옵션 매수
    - 동적자산배분 : 주식 매수, 채권 매수
  - 보장치(최저가치), 완충치(포트폴리오 현재가치 - 보장치), 승수(위험자산 투자 배수), 노출치(완충치 * 승수)
  - CPPI : 최초 보장치가 무위험이자율로 증가
  - TIPP : 최고의 보장치 유지, 하락,횡보장서 유리

7. 파생상품펀드 주요 내용
  - 시장중립형 : 주식형차익거래, 인덱스차익거래, 합병차익거래, 전환사채차익거래
  - 구조화형(델타헤징) : 등락이 반복되는 장세에서 수익
  - 시스템운용형 : 모멘텀(단기), 역발상(장기), 변동성관리
    - 추세장에서 성과 양호, 박스권,등락장에서 부진

8. 선물의 균형가격
  - 주가지수 선물 : F=S*[1+(r-d)*t/365]
  - 통화선물 : F=S*[1+(r-r*)*t/365]
  - 선물가격 상승 요인 : 현물가격 상승, 이자율(원화이자율)상승, 배당수익률(외화이자율) 하락, 잔존만기일수 증가
  - r > d, F > s, 정상시장(콘탱고), b > 0
  - r < d, F < S, 역조시장(백워데이션), b < 0
  - 매수차익거래 : 이론F < 실제F, F매도, S매수
  - 매도차익거래 : 이론F > 실제F, F매수, S매도

9. 선물 헤지거래
  - 매도헤지 : 선물매도, 가격하락 위험 헤지
  - 매수헤지 : 선물매수, 가격상승, 위험 헤지
  - 제로베이시스 : 만기 청산, 베이시스위험 헤지
  - 랜덤베이싀스 : 만기 전 청산, 시장위험 헤지
  - 헤지비율 : 헤지를 위함 선물포지션 계약수 산정

10. 옵션 스프레드전략 : 수직스프레드
  - 강세(Bull)
    - 낮은 행사가격 매수, 높은 행사가격 매도
    - 콜강세 초기 순지출, 풋강세 초기 순수입
  - 약세(Bear)
    - 낮은 행사가격 매도, 높은 행사가격 매수
    - 콜약세 초기 순수입, 풋약세 초기 순지출
  - 레이쇼버티칼
    - 매수 계약수 < 매도 계약수
    - 콜레이쇼버티칼 : 횡보/하락장에서 우수
    - 풋레이쇼버티칼 : 횡보/상승장에서 우수
  - 백스프레드
    - 매수 계약수 > 매도 계약수
    - 콜은 상승 시, 풋은 하락 시 이익 최대
  - 스트래들
    - 동일한 행사가격의 콜과 풋을 동시에 매매
    - 변동성 확대 시 매수, 변동성 축소 시 매도
  - 스트랭클
    - 낮은 행사가격 풋, 높은 행사가격 콜 매매
    - 스트래들보다 초기 투자비용 절약
    - 변동성 확대 시 매수, 변동성 축소 시 매도
  * 옵션프리미엄 = 행사가치 + 시간가치

11. 옵션 민감도 지표
  - 델타 : 0<Call<1, -1<Put<0
  - 감마 : 항상 양(+) 값, 등가격에서 최대
  - 세타 : 시간가치 감소에 따라 손실, 항상 음(-)의 값
  - 베가 : 변동성 증가 시 가치 확대, 항상 양(+)의 값
  - 로 : 금리상승 시 콜옵션 호재(+), 풋옵션 악재(-)

12. 주요 계약
  - 금리선도계약 : 매입자(차입자), 매도자(대여자)
    - 자금 인수도 없이, 약정금리와의 차액만 결제
  - 금리스왑 : 금리 간 차액만 현금결제
    - 금리상승 예상 시 : 고정금리 지급포지션
    - 금리하락 예상 시 : 고정금리 수취포지션
  - 통화스왑 : 원금이 교환되는 Funded Swap
  - CDS 프리미엄 : 만기 길수록, 채무불이행 가능성 높을수록, 보장매도자 신용 높을수록, 상관관계 낮을수록, 준거자산 회수율 낮을수록 상승

13. 이색옵션
  - 경로의존형

    - 룩백옵션 : 룩백콜 최저가격, 룩백풋 최고가격

14. 파생상품펀드의 리스크관리
  - 파생결합증권 발행사

## 재3과목 부동산펀드
1. 부동산펀드 법적형태 7가지
